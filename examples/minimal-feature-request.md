# Request: Add undo/redo support

I want the editor to support undo and redo.

Requirements:

- Undo should restore the previous editor text value.
- Redo should restore the value after undo.
- Cmd/Ctrl+Z should undo.
- Cmd/Ctrl+Shift+Z should redo.
- Add tests for core behavior.
- Do not change unrelated editor behavior.
